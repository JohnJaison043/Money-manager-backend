# Money Manager Backend

Backend server for the Money Manager application built with Express.js and SQLite.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file in the root directory (if needed).

## Running the Server

```bash
npm start
```

The server will run on port 5000.

## Features

- Transaction management (income, expense, transfer)
- Division-based tracking (office, personal)
- RESTful API endpoints
- SQLite database
